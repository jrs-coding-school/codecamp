# Basic Math Functions

Is a library that contains common math functions, mainly for practicing function composability.

## Demo

```
const { add } = require('basic-math-fn')
add(1,2)
=> 3
```

## Install

`npm install basic-math-fn`

## Api

- add(a,b)
- subtract(a, b)
- multiply(a, b)
- divide(a, b)
